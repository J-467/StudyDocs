Study Docs

Study Docs is a web application that serves as a centralized resource hub for academic classes at Yale.


This web app provides resources that offer in-depth explanation to concepts within each class.


This web app has a list of some majors in Yale.


On selecting the desired class, there are some resources including the syllabus to that class from previous years.


There are also links to vidoes to explain concepts better.


The resources are ideally to be submitted by students who have taken that class in the past.
